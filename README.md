# LiveSoccer
Application displaying football teams and their upcoming matches using https://www.sportmonks.com/ API

The project contains:  
- Kotlin Coroutines
- Retrofit 2
- Navigation of fragments
- Adaptive interface
- ViewModel and LiveData
- OneSignal integration
- WebView
